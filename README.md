
# Jornada de Aprendizado e Desenvolvimento Técnico

Este repositório documenta minha evolução em programação e análise de dados, focando na aplicação dessas ferramentas no contexto da Física Médica. Meu objetivo é demonstrar como utilizo a tecnologia para otimizar processos laboratoriais e análises estatísticas.

---

## 📚 Tecnologias em Desenvolvimento

### 🔹 C++ e Instrumentação
Utilizo C++ principalmente para o desenvolvimento de sistemas embarcados com **Arduino**. 
- **Foco:** Automação de coleta de dados físicos e controle de dispositivos de medição.
- **Destaque:** Implementação de algoritmos para levantamento de curva característica de componentes eletrônicos (Resistores), integrando hardware e software para maior precisão experimental.

### 🔹 Linguagem R e Estatística
Em fase de aprofundamento para dar suporte a análises bioestatísticas e modelagem de dados clínicos.
- **Objetivo:** Manipulação de grandes bases de dados e geração de gráficos de alta qualidade para relatórios científicos.

---

## 📈 Linha do Tempo de Projetos

### [2026.1] - Caracterização de Instrumentos (Física Experimental III)
* **Contexto:** Projeto acadêmico na USP para medir resistência e curvas de tensão/corrente.
* **Ação:** Desenvolvimento de código em C++ para ler sensores via Arduino e exportar dados brutos.
* **Resultado:** Redução do erro humano na coleta manual e maior agilidade no tratamento dos dados no Excel/BI.

### [Em andamento] - Domínio de Estruturas de Dados
* Estudo de lógica de programação aplicada à física de radiações e processamento de sinais.
